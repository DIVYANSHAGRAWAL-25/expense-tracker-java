# Expense Tracker (Java CLI based Project)

## Project Description

this is a simple basic command line (CLI) based project using java programmind language. The purpose of this project is to make an app which is not internet based once it is installed and they can put their daily expenses in the organinzed manner. 

the user can add their expenses, view all saved / recorded expenses, calculate the total spending, and delete the unwanted and faulty entries and store the entries in a file for future reference and clear calculation.

---

##  Features

1 add new records / expenses
2 view all records / expenses
3 calculate the total expenditure
4 delete or remove an expense
5 save and load data by file handling
6 simple CLI user friendly interface

---

##  Technologies Used

1 Java
2 Object-Oriented Programming
3 Array list
4 File Handling (Scanner, PrintWriter)
5 Exception Handling

---

##  How to Run and Use the Project

### 1 Open terminal and reach to the directory of the project

cd path/to/ExpenseTracker

### 2 Compile the program by the below command

javac Main.java

### 3 Run the program by the below command

java Main

---

##  Project Structure

1 Main.java will handles user interaction
2 expense.java will represents an expense
3 expenseRecords.java will manages all expenses
4 expenseFileHandler.java will handles file operations
